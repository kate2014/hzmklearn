唯一优品品控

一、唯一优品网商品质量要求和规范

      为了维护唯一优品网平台的声誉和品牌，唯一优品网严格要求上线销售的商品是正品真货，符合国家相应法律法规、标准要求，对于不符合要求的商品一律不予销售，因此为了防止不合格商品的上线销售，在专场上线前，唯一优品网将对以下商品进行抽检检验和监督，如果商品抽检不合格，依据严重程度，进行相应处罚。

      同时为了防止商家实际销售商品与送检商品不一致，唯一优品网将不定期对已上线销售商品神秘购买进行抽检，且会从部分买家退货中抽取一定量的商品进行售后监督，如果商品抽检发现不合格，将严厉处罚。

二、唯一优品网抽检商品范围

所有在唯一优品网销售的商品，包括但不限于以下类目及商品：

童鞋：儿童皮鞋，休闲鞋，婴幼儿布鞋等；

童装：儿童服装，内衣，睡衣，羽绒服，皮革服装，羊绒衫等；

玩具：塑料玩具，积木玩具，毛绒玩具，仿真玩具，电动玩具，婴幼儿推车，学步车，儿童三轮车，儿童自行车等；

母婴商品：孕妇服装，婴幼儿服装，纸尿裤、产妇卫生巾、湿巾，纸尿裤、产妇卫生巾、湿巾，护肤用品等。

三、商家寄样规则

  • 1、 商家提报某品牌的专场活动后，唯一优品网将通过消息中心及邮件通知商家寄送样品进行检验，样品仅供审核参考，裁定商品是否后续可以上活动。请不要误认为邮递样品或者缴纳保证金后就一定能上活动，最终解释权归唯一优品网所有。 寄送数量的设定：按专场商品款数的20%邮寄样品（同材质、同规格商品可不计入专场商品总款数），最低不低于2款，最高不超过10款。每款商品需提供1件样品。特殊商品的处理方法：童车、童床、汽座等大体积商品，寄送对应商品的质检报告（复印件带公章）即可。进口类商品需提供报关单及出入境检验检疫报告（复印件带公司红章）。
  • 2、 邮递样品时，请添加1张包含寄样商品详情的邮寄单，样品邮寄单将在通知需邮寄样品时提供。寄送地址：浙江省杭州市江干区九盛路9号东方电子商务园12幢6楼，收件人：唯一优品网品控部
  • 3、 样品一经发出不予退还，快递费用由商家承担，不支持到付。
  • 4、 样品收取后，由唯一优品网统一保管，且有权自行处理。
  • 5、 商家需要收到寄样通知的3个工作日内寄出样品，同时提交快递单号，否则将无法上线专场。

    • 四、品控检验环节

    1、样品检验

          商家提报品牌专场或优品惠活动时，唯一优品网将要求商家寄送2-10件样品进行检验，商家需在收到抽检要求（邮件+消息中心）的3个工作日内，将相应商品邮寄至唯一优品网，并回复邮件提供正确的快递信息。若有问题，请在3个工作日内及时回复邮件。逾期未收到反馈，视为不配合，一直冻结公司特卖资格，直至寄样检测成功。

          每一位新入驻的商家或者商家新增加了品牌，均需进行样品抽检，单件商品按照下述表格中的“检测项目”进行检测，出现任一项不合格情况，该商家将无法入驻唯一优品网。

    2、神秘购买检验

          唯一优品网品控将随机对在线销售的商品进行神秘购买，然后根据“检测项目”进行检测，出现其中一项不合格的情况，将根据《唯一优品网商家违规处罚规则》进行扣分处罚。

    3、退货抽检

          抽检产生退货商家，选择若干买家退货要求买家将商品以及填写完整的《退货单》寄到唯一优品网，根据“检测项目”进行检测，出现其中一项不合格的情况，将根据《唯一优品网商家违规处罚规则》进行扣分处罚。原则上退货商品统一由唯一优品网保管，作为退货抽检证据保存。

    六、申诉途径

          若商家对抽检结果有质疑，可在3个工作日内提交申述以及相关质检合格证明至唯一优品网进行申诉，逾期不受理。

          唯一优品网将针对商家申诉时提交的质检合格证明判断是否需要复检。

    （一）根据质量指标的特殊性及检测商品或部位的可重现性，以下情况无法复检：
  • 1、超过期限提起申诉的；
  • 2、无备样或者备样与原样存在明显的差异（如颜色不同或款式不同）而无法在原样上进行复验的；
  • 3、商家提出复检时，商品在复检有效期内于正常贮存条件下已失效的；
  • 4、甲醛含量检验结果为不合格的；
  • 5、法律、法规规定不得复检的其他情况。
    • （二）原则上，除上述无法复检的情形或商家提交的申诉质检证明存在作假情形外，唯一优品网一律支持复检。
    （三）对不合格项目进行复检时，可以在原样上检验的在原样上复检，不可以在原样上检验的采用备用样复检。
    （四）复检结果仍不合格的，维持原检验结果不变，复检费用由商家自行承担（优先从货款中扣除，若无货款需商家补交）；复验结果发生变更的，以复检结果为准，复检费用由唯一优品网承担。
