对账/结算

何时开始对账？

  • 每场销售结束3天后，商家管理后台自动生成当期结算单，即可开始对账。
  • 专场销售后由于退货等情况，结算金额会有变化，最终结算金额以结算日的金额为准。

如何对账？

  • 请核对商家管理后台的销售数据，确认各金额与明细是否一致，如对结算数据有异议，请于及时向唯一优品网提出，逾期则视为商家对结算数据无异议，唯一优品网在结算日期后自动转入商家的结算账户，商家可从结算账户申请提现。

结算金额

结算金额 = 订单总额 - 退货总额 - 赔款总额 - 佣金

  • 赔款金额包含运费赔偿、额外赔偿等
